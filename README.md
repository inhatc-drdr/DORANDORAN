# drdr-node

### 프로젝트 구조
```
DEMO-NODE
⊢ public    // 클라이언트 폴더
∣   ⊢ js
∣   ⊢ css
∣   ⊢ imgs
⊢ models    // DB 관련 스키마, 연결 코드
⊢ routes    // 라우터
⊢ views     // 템플릿
⊢ app.js    // 서버
⊢ package.json 
⊢ nodemon.json
⊢ babel.cofing.json
⊢ .env      // 환경변수, 비밀번호 등 보호되어야할 정보들이 담긴 파일
⊢ readme.md
⊢ readmes   // 설명 파일 등
```

### How to use
1. NPM 모듈 설치
```
npm install
```

2. 실행
```
npm run dev
```
