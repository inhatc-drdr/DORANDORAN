# drdr-node

### 프로젝트 구조

```
DEMO-NODE
⊢ models    // DB 관련 스키마, 연결 코드
⊢ config    // crypto, passport 등 모듈
⊢ routes    // 라우터
⊢ app.js    // 서버
⊢ package.json
⊢ nodemon.json
⊢ babel.cofing.json
⊢ .env      // 환경변수, 비밀번호 등 보호되어야할 정보들이 담긴 파일
⊢ readme.md
⊢ readmes   // 추가 readme 파일
```

### How to use

1. NPM 모듈 설치

```
npm install
```

2. 실행

```
npm run dev
```
