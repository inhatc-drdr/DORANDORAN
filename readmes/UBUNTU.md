## How to USE

### Git Pull
```
sudo git pull origin main
```

### Start
```
nohup node app.js &

exit
```

### Port Kill
```
sudo fuser -k -n tcp 5000
```

### Install Nodejs
```
sudo apt update
```

```
sudo apt install nodejs

```
2. Install NPM
```
sudo apt install npm
```
